# ldmap

<!-- badges: start -->

<!-- badges: end -->

The goal of ldmap is to simplify common operations on statistical
genetics and genomics datasets by providing a set of simple, performant
classes for dealing with (human) genomic annotation data and GWAS
summary statistics.
